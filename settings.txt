Перед запуском программы необходимо установить необходимые библиотеки с помощью файла с зависимостями requirements.txt
Для этого сначала необходимо перейти в директории с проектом, затем выполнить команду в командной строке "pip install -r requirements.txt"
Теперь, после успешной установки для запуска приложения, необходимо запустить файл main.py, с помощью команды
"python main.py"
